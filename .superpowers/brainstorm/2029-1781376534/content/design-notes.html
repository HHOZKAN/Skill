Hooks React — Notes
Les hooks permettent d'utiliser l'état et d'autres fonctionnalités de React sans écrire de classe.
État local
useState retourne un tableau — [valeur, setter]. Chaque appel au setter déclenche un re-render.
javascript
const [count, setCount] = useState(0);
const [user, setUser] = useState(null);
💡 Le setter est stable entre les renders — pas besoin de le mettre dans les dépendances useEffect.
Effets de bord
Se lance après chaque render (par défaut)
Le tableau de dépendances contrôle les re-runs
Retourner une fonction = cleanup automatique
« Chaque render est un snapshot — les effets voient la valeur au moment du render. »
